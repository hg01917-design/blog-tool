# 올인원 자동화 도구 통합 설계서
> 기존 Flask 웹앱에 메뉴 추가해서 올인원으로 확장

---

## 전체 메뉴 구조

```
🏠 메인 대시보드
├── 📝 블로그 자동화 (개인)
├── 🛍️ 쇼핑몰 관리 (개인)
│     ├── 상세페이지 자동 생성
│     ├── 주문 / 송장 관리
│     └── 재고 관리
├── 📊 매출 대시보드 (회사)
├── 📋 사장님 보고서 (회사)
└── 📚 아이 학습 앱 (개인)
```

---

## 메인 대시보드
한눈에 볼 수 있는 요약 화면:
- 오늘 블로그 발행 현황
- 오늘 쇼핑몰 주문 수
- 미처리 주문 수
- 재고 부족 상품 수
- 이번달 매출 요약
- 아이들 오늘 학습 완료 여부

## 기술 스택
- **서버**: 기존 Flask (Python)
- **배포**: 워드프레스 하위도메인
- **DB**: 구글 시트 (블로그 결과물 저장)
- **프론트**: HTML / CSS / JS
