# 블로그 자동화 도구 업그레이드 작업 지시서
> 현재 환경: Flask 웹앱 (워드프레스 하위도메인에 배포)

---

## 1. 키워드 자동 수집
- 구글 자동완성 크롤링
- 구글 연관검색어 크롤링
- 네이버 자동완성 크롤링
- 네이버 연관검색어 크롤링
- 네이버 데이터랩 API 연동
- 수집된 키워드 중 중복/유사한 것 자동으로 묶거나 하나만 선택
- 정제된 키워드 목록을 파일로 저장 or 화면에 표시

---

## 2. 네이버용 이미지 자동 생성
- 키워드 입력하면 Unsplash API로 여행 관련 사진 자동 검색
- 사진 위에 제목 텍스트 오버레이 (Canvas로 합성)
- 이미지 파일로 다운로드 가능하게
- 네이버 포스팅 복붙할 때 바로 쓸 수 있게

---

## 3. 제휴링크 DB 연동
- 키워드/지역/숙소유형별 제휴링크 DB 구축 (구글시트 or JSON)
- 예시 구조:
  - 키워드: "방콕 호텔"
  - 제휴사: 세시간전 (아고다)
  - 링크: https://...
  - 카테고리: 동남아
- 콘텐츠 생성 시 키워드에 맞는 제휴링크 자동 매칭해서 본문에 삽입
- 포스팅 하단에 제휴 고지문 자동 삽입:
  > "이 포스팅은 제휴 마케팅이 포함된 글입니다. 링크를 통해 예약·구매 시 운영자에게 소정의 수수료가 지급될 수 있으며, 이는 독자에게 추가 비용을 발생시키지 않습니다."

---

## 4. SNS 자동 발행
워드프레스 자동발행 완료 후 아래 플랫폼에 동시 발행:
- 페이스북 (Meta Graph API)
- 스레드 (Threads API)
- 인스타그램 (Instagram Graph API) - 이미지 필수
- 핀터레스트 (Pinterest API) - 백링크 효과

각 플랫폼별 후킹 문구 AI로 자동 생성 후 발행

---

## 5. 백링크 모니터링
- 백링크 등록한 URL 목록 관리
- "백링크 체크" 버튼 누르면 한번에 전체 확인
- 살아있음 ✅ / 지워짐 ❌ 한눈에 표시
- 서버 비용 없이 버튼 클릭 시에만 체크하는 방식

---

## 6. 수익 대시보드 (반자동)
- 애드센스 / 쿠팡파트너스 / 세시간전 수익 입력칸
- 월 목표 금액 설정 (목표: 100만원)
- 달성률 게이지로 표시
- 완전자동 연동은 어려우니 숫자 직접 입력 → 한눈에 보기

---

## 7. 글감 메모장
- 폰에서도 접근 가능한 간단한 메모 기능
- 아이디어 입력 → AI가 자동으로 키워드로 변환
- 블로그 작성기에 바로 연결

---

## 8. 시즌 키워드 알림
- 월별 시즌 키워드 DB 미리 구축
  - 예: 3월 → 벚꽃, 봄여행 / 7~8월 → 여름휴가, 해외여행
- 이번달/다음달 추천 키워드 표시
- "이 키워드 미리 써두세요" 알림

---

## 9. 경쟁 블로그 반자동 분석
- 1위 블로그 URL or 내용 붙여넣기
- AI가 분석해서 "내 글에 추가하면 좋을 것들" 추천
- 완전자동 크롤링은 어려우니 반자동으로 구현

---

## 10. 결과물 자동 저장 (구글 시트)
- 글 생성 완료되면 구글 시트에 자동 저장
- 저장 항목:
  - 날짜
  - 키워드
  - 제목 (후보 3개)
  - 본문 내용
  - 제휴링크
  - 발행 여부 (미발행/발행완료)
- "내 글 목록" 탭에서 저장된 글 불러오기
- 폰에서도 구글 시트로 확인 가능
- 제휴링크 DB도 같은 시트에서 관리

---

## 11. 재활용 기능
- 잘 됐던 글을 살짝 변형해서 다른 플랫폼에 재발행
- 티스토리 잘 된 글 → 워드프레스 변형 재발행
- AI가 자동으로 리라이팅

---

## 11. 여행 블로그 제목 설계 프롬프트
블로그 작성기에 키워드 입력하면 AI가 아래 규칙으로 제목 자동 생성:

**제목 공식**
- 숫자형: "[키워드] [숫자]가지 총정리"
- 손해형: "[키워드] 이거 모르면 손해"
- 공감형: "[키워드] 막막하죠? 이렇게 하세요"
- 의외성: "[키워드] 意外로 이런 이유가 있었다"
- 경비형: "[키워드] 경비 총정리 [숫자]만원 아끼는 법"

**규칙**
- 롱테일 키워드 그대로 제목에 포함
- 30자 내외
- 궁금증/손해/혜택 중 하나 반드시 포함
- 숫자 들어가면 클릭률 높아짐
- 제목 후보 3개 뽑아서 하나씨가 고르게

**첫 문단 공식**
- 독자 상황 공감 + 해결될 거라는 암시
- 예시: "방콕 호텔 알아보다 가격에 놀라셨죠? 저도 그랬어요. 근데 이거 알고 나서 완전 달라졌어요."

---

## 현재 플랫폼별 발행 구조
| 플랫폼 | 방식 | 제휴링크 |
|---|---|---|
| 워드프레스 | 완전 자동 발행 | 세시간전 + 쿠팡 |
| 티스토리 | 웹에서 뽑아서 복붙 | 세시간전 + 쿠팡 |
| 네이버 | 웹에서 뽑아서 복붙 | 쿠팡만 (텍스트 URL) |
